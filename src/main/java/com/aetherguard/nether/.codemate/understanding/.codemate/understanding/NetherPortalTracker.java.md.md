NetherPortalTracker is a utility class for monitoring players' Nether portal usage on a Minecraft server. It uses a thread-safe map to store PortalProfile instances, each tracking the number of times a player has teleported through a Nether portal. The class provides methods to record portal teleport events per player and analyze their usage patterns.
